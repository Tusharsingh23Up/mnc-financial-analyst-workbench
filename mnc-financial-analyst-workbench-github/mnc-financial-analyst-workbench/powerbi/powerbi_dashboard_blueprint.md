# Power BI Dashboard Blueprint

Build four report pages:

## 1. Executive Summary
Cards:
- Revenue
- Revenue CAGR
- EBIT margin
- FCF margin
- DCF value per share

## 2. Financial Performance
Visuals:
- Revenue trend
- EBIT and net income trend
- Margin trend
- FCF trend

## 3. Peer Benchmarking
Visuals:
- EBIT margin comparison
- Revenue CAGR comparison
- Debt-to-cash comparison
- FCF margin comparison

## 4. Valuation and Scenario Analysis
Visuals:
- DCF valuation bridge
- Sensitivity table
- Scenario selector
- Recommendation card
